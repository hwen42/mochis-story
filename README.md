# Mochi's Story

A family tree and photo timeline for Mochi, an Akita.

## Updating the website

The website is hosted with GitHub Pages. Edit `index.html`, `styles.css`,
`app.js`, or files in `assets/`, then commit and push the changes to GitHub.
GitHub Pages will publish the updated version automatically.

Timeline memories added through the website are stored in that browser on that
device. They are not uploaded to GitHub or shared between devices.
